# Prompt: Refactoring

```
Propose a refactor for this code.
Constraints:
- No external behavior/contract change (if one is needed, this is an architecture-change, not a refactor).
- Confirm test coverage exists first; propose characterization tests if missing.
- Prefer extracting a named pattern (document in .claude/patterns/) over a one-off cleanup if the shape repeats elsewhere in the codebase.
- Keep steps small and independently buildable/testable.
Report: current duplication/coupling problem, proposed shape, migration steps in order.
```
