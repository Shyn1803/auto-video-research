# Prompt: Feature Design

```
Design [feature] against:
- docs/SRS.md for the functional requirement it serves (cite FR-XX).
- docs/glossary.md for correct domain terminology — don't invent a synonym.
- .claude/context/architecture.md for where it fits in the pipeline/module layout.
- The Layout Engine boundary if it touches storyboard/scene/prompt code — AI content only, never layout.
Produce: a feature spec using .claude/templates/feature-spec.md, explicit UI states (default/loading/empty/error/disabled),
and a note on what existing pattern (.claude/patterns/) it can reuse vs. what's genuinely new.
```
