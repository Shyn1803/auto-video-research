# Prompt: UI Review

```
Review this screen/component against docs/design/wireframe.html and docs/design/design-system.md.
Check:
- Matches the wireframe's layout contract for this screen (or has a documented, approved deviation).
- Uses existing design-system.md components/tokens rather than inventing new ones ad hoc.
- Covers all 5 UI states where applicable: default/loading/empty/error/disabled, plus accessibility.
- If this is the Phân cảnh or Hoàn thiện screen: confirm correct composition used — Scene (per-scene, Phân cảnh) vs Video (whole-video preview only, Hoàn thiện) — see .claude/patterns/scene-video-composition-split.md.
Then: actually run it in a browser and exercise the golden path + at least one edge case before calling it done.
```
