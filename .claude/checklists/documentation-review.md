# Checklist: Documentation Review

- [ ] No duplicated fact — every claim traces to exactly one canonical doc
- [ ] No contradiction with `docs/glossary.md` terminology
- [ ] Cross-references added both directions where relevant
- [ ] `docs/README.md` reading-order table updated if a new top-level doc was added
- [ ] If this changes a product/architecture fact in `docs/`, it was proposed for user (PO) confirmation, not unilaterally finalized
- [ ] `.claude/` files link to `docs/` rather than restating detailed facts
