# Checklist: Accessibility Review

- [ ] All 5 UI states covered: default/loading/empty/error/disabled
- [ ] Keyboard navigable (stepper, scene editor, drawer components)
- [ ] Color contrast meets WCAG AA for text over theme backgrounds (relevant given per-theme accent color system in video-taste.md)
- [ ] Interactive elements have accessible names (not icon-only with no label/aria)
- [ ] Video preview player controls are screen-reader operable
- [ ] Vietnamese text renders correctly (diacritics) across all target fonts/themes
